# 🧙 Pythonwarts

## ⚡ Pythonwarts: The Witch Programming Tournament
Welcome to the magical world of Pythonwarts, a Python programming quiz entirely set in the Harry Potter universe!
This project was developed in partnership by myself and my classmate José Roberto as an assessment requirement for the Introduction to Programming course in our Information Systems program at the Federal Rural University of Pernambuco (UFRPE).

## 🔮 About the Project
Pythonwarts is a CLI (Command Line Interface) question and answer game designed to test the programming logic knowledge of witches and wizards. The project was built from scratch using purely the Python language, applying fundamental concepts such as:
- Variables and Data Types
- Conditional Structures
- Repetition Structures
- Functions and Modularization

## 🛠️ Prerequisites
Before you can start coding, you will need to have the following installed on your machine:
- Python 3.x installed.
- A code editor such as VS Code, or your system's terminal.

## 🚀 How to Play
Follow the steps below to begin your journey in Pythonwarts:
1. Download or clone this repository to your machine.
2. Open the terminal in the folder where the project files are saved.
3. Run the initialization spell with the command below:

```bash
main.py
```
